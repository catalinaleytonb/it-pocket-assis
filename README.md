# IT Pocket Tool 🛠️ 📱

**IT Pocket Tool** es una aplicación móvil nativa para Android diseñada para optimizar las tareas diarias de técnicos e ingenieros de soporte TI. Este proyecto combina lógica de redes, gestión de reportes y una interfaz moderna basada en Material Design 3.

## ✨ Características Principales

- **Calculadora de Subredes:** Cálculo preciso de máscaras, rangos de IP y prefijos de red (CIDR).
- **Gestor de Reportes Técnicos:** Formulario dinámico para registrar fallas y soluciones, con capacidad de exportar y compartir archivos `.txt` mediante `FileProvider`.
- **Modo Oscuro Adaptativo:** Interfaz que respeta la configuración del sistema para mejorar la visualización en entornos de trabajo.
- **Arquitectura Robusta:** Uso de `ViewBinding` para un manejo eficiente de la UI y `JUnit` para pruebas unitarias de la lógica de red.

## 🛠️ Stack Tecnológico

- **Lenguaje:** Java ☕
- **Plataforma:** Android SDK (API 34+)
- **Diseño:** XML con Material Design 3
- **Componentes Clave:** - `ViewBinding` para vinculación de vistas.
  - `Intent` & `FileProvider` para interoperabilidad entre apps.
  - `JUnit` para validación de algoritmos de subredes.

## 🚀 Instalación y Uso

1. Clona este repositorio:  
   `git clone https://github.com/catalinaleytonb/it-pocket-assis.git`
2. Abre el proyecto en **Android Studio**.
3. Sincroniza los archivos de **Gradle**.
4. Ejecuta la aplicación en un emulador o dispositivo físico.
